# Backend — Cafe Management API

Spring Boot REST API chạy trên cổng **8080**.

## Yêu cầu
- Java 17+
- MySQL đang chạy với database `cafe_management` (user: `root`, pass: `achiendz`)

## Chạy
```bash
./mvnw spring-boot:run
```

API sẵn sàng tại: http://localhost:8080/api
